Final Project - Spotify Musicapp

How to install and run locally:
1. Unzip the "Final Project" folder containing the Spotify-client and Spotify Server folders

2. Run app.js in node to start the server

3. Run main.py with default settings (Press enter when first prompted)
      Additionally, you may skip step 2 if you run main.py with ec2-client-config.ini
      and use AWS Elastic Beanstalk

4. Create a user and use app

Done with set up!
